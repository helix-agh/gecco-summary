# GECCO Accepted Papers Explorer

A small, fast web app for exploring the accepted papers of [GECCO 2026](https://gecco-2026.sigevo.org/) — the Genetic and Evolutionary Computation Conference. Browse every accepted full paper by track, author, and institution, see what's popular, and jump straight to the ACM Digital Library.

## How it works

- **Data** lives in `src/data/papers.json`, generated by `scripts/scrape.py` (Python, standard library only). The script parses the [accepted papers list](https://gecco-2026.sigevo.org/Accepted+Papers) and matches each title to its ACM DOI through the open [Crossref API](https://api.crossref.org) (exact match first, then a conservative fuzzy match for titles that changed slightly in camera-ready).
- **Frontend** is a static React + TypeScript app built with Vite — no backend, the dataset ships with the bundle.
- **Deployment** is GitHub Pages via GitHub Actions on every push to `main`.

## Development

```sh
npm install
npm run dev        # start the dev server
npm run lint       # eslint (type-checked rules)
npm run format     # prettier
npm run build      # typecheck + production build
npm run scrape     # refresh src/data/papers.json (requires Python 3.10+)
```

## Data notes

- Affiliations are kept as published. Multiple affiliations per author are split on commas, which occasionally splits a single institution name (e.g. "University of California, Irvine") — a normalization pass is planned.
- Institution counts mean "papers with at least one author from that institution", counting each paper once.
